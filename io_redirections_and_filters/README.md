# Shell, I/O Redirections and Filters

## Description
This project introduces shell redirections and filters.  
You will learn how to handle standard input and output, and how to use useful commands like `cat`, `grep`, `sort`, `tr`, and more.

## Scripts

### 0-hello_world
Prints “Hello, World” followed by a new line to the standard output.

## Requirements
- All scripts are exactly two lines long
- Must end with a new line
- Must be executable
- Must start with `#!/bin/bash`
- No use of backticks, `&&`, `||`, or `;`
- Tested on Ubuntu 20.04 LTS
